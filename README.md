## 更新時の私へ
gitはC\:直下のフォルダに移動したため、Googleドライブ内のファイルを編集しただけでは同期は出来ていません。
辞書やファイルなどを編集し、アップロードした際は手動で同様の構造をC\:git\lineparine以下のディレクトリに導入を行わなければなりません。
もし忘れているかもしれないと思ったら、ふぁふすにこっそり教えてあげて下さい！
